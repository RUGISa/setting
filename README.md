# 설정서랍 데스크톱 앱

세계관 설정을 카드, 폴더, 타임라인, 관계도로 정리하는 Electron 앱입니다.

## 실행 방법

터미널에서 이 폴더로 이동한 뒤 실행하세요.

```bash
npm install
npm start
```

## 설치 파일 만들기

Windows 설치 파일:

```bash
npm run build:win
```

Mac 설치 파일:

```bash
npm run build:mac
```

빌드 결과는 `dist` 폴더에 생성됩니다.

## 저장 방식

이번 버전은 실제 파일 저장을 지원합니다.

- 저장: 현재 열려 있는 `.drawer` 파일에 저장합니다. 처음 저장할 때는 저장 위치를 묻습니다.
- 내보내기: 다른 이름으로 `.drawer` 파일을 저장합니다.
- 가져오기: `.drawer` 또는 `.json` 파일을 엽니다.
- 메뉴 단축키:
  - Ctrl/Cmd + S: 저장
  - Ctrl/Cmd + Shift + S: 다른 이름으로 저장
  - Ctrl/Cmd + O: 열기

앱 내부에는 localStorage 백업도 남기지만, 중요한 설정은 `.drawer` 파일로 저장해두는 것을 추천합니다.


## 관계도 연결 방식

관계도에서 노드 두 개를 연결하려면 첫 번째 노드를 `Shift + 클릭`, 두 번째 노드도 `Shift + 클릭`하면 됩니다.

기존 `선 연결` 버튼도 남아 있지만, 기본 사용은 `Shift + 클릭` 방식이 더 빠릅니다.


## 관계도 새 연결 방식

관계도는 제목만 보이고, 노드를 클릭하면 설명이 펼쳐집니다.

선을 연결하려면:

1. 첫 번째 노드를 Shift + 클릭
2. 두 번째 노드를 Shift + 클릭
3. `선 연결` 버튼 클릭
4. 선 이름 입력 후 저장

선은 각 노드 박스의 정가운데를 기준으로 연결됩니다.
